# Домашнее задание 2

**Тема:** AI

## Цель

Научиться делать AI в Unreal Engine. Попробовать различные реализации и подходы в разработке AI.

## Задача

Добавить в проект AI.

### NPC

- Добавить простой NPC.
- Он должен патрулировать определённую область.
- Иметь возможность взаимодействия с игроком (например, диалог или выдать какой-то предмет).
- После отхода игрока должен возвращаться на маршрут патрулирования.

### Противник

- Добавить противника.
- Противник должен охранять определённую область.
- При заходе в область игрока бот должен переходить в состояние атаки и атаковать игрока.
- Если игрок уходит из области, через некоторое время бот должен снова перейти в состояние охраны.

### Задание со звёздочкой

Добавить несколько разных типов противников (например, ближнего и дальнего боя). Соответственно, их поведение должно отличаться.

### Альтернатива: босс-файт

Вместо предыдущих пунктов есть опция сделать босс-файт:

- Босс должен иметь много HP.
- Должно быть несколько этапов боя.
- Поведение босса во время этих этапов должно отличаться.
- Урон боссу можно наносить не только атаками игрока, но и ловушками.

> За домашку можно получить доп. баллы, если сделать оба пункта.
