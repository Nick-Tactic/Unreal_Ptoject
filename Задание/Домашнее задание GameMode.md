# Домашнее задание 3

**Тема:** GameMode

## Цель

Научиться работать с GameMode, GameState, PlayerState.

## Задача

Создать логику игры и прохождения уровня.

- Продумать логику игры.
- Реализовать необходимые акторы и классы для игры (pickables, equipables, enemies).
- Создать GameMode. Сделать спавн игрока и начало игры после нажатия кнопки Start в UI.
- Сделать респавн игрока после смерти.
- Добавить контрольные точки для респавна игрока.
- Продумать условия для прохождения и завершения уровня.
- Добавить **GameState**. Тут должны храниться и считаться достижения, необходимые для прохождения уровня (количество собранных предметов и т.д.).
- Добавить **PlayerState**. Тут должна храниться персональная статистика игрока. Она должна отображаться по нажатию клавиши (например, Tab).
- По завершению уровня должна останавливаться игра и выводиться сообщение о завершении.
